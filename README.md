## Run Locally

Clone the project

```bash
  git clone https://github.com/procsspace/leafy-bot
```

Go to the project directory

```bash
  cd leafy-bot
```

Install dependencies

```bash
  npm install
```

Deploy Commands

```bash
  node ./deploy.js
```

Start The bot
```bash
  npm run dev
```
